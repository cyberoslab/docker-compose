# What is this

  A dockerlized docker-compose image for linux/arm64, linux/arm, linux/amd64.

# How to use this image

### Install docker-compose as a container

```
$ sudo curl -L --fail https://github.com/cyberoslab/docker-compose/releases/download/latest/run.sh -o /usr/local/bin/docker-compose
$ sudo chmod +x /usr/local/bin/docker-compose
```
or
```
$ sudo curl -L --fail https://github.com/cyberoslab/docker-compose/releases/download/1.29.2/run.sh -o /usr/local/bin/docker-compose
$ sudo chmod +x /usr/local/bin/docker-compose
```
